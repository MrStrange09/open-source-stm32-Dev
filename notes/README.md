# Notes Index

This folder is for learning notes attached to working checkpoints.

`Info.md` stays as the current status page. These notes explain what changed,
why it changed, and what to try next.

Suggested habit for each checkpoint:

1. Make the feature work.
2. Add or update one note file for that feature.
3. Commit code and notes together.

Current notes:

- `blink.md`: GPIO output and delay-loop blink.
- `systick-blink.md`: SysTick-based non-blocking blink.
- `pwm.md`: Timer PWM fade on the onboard LED.
- `base-bootloader.md`: base bootloader image and app layout.
- `uart-driver-1b.md`: first UART driver on USART2 with an interrupt echo.
- `project-structure.md`: why headers, source files, and the Makefile are split.
